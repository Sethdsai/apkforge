how it works

1

drop your html

paste or type your entire html code into the editor. inline css and js all work fine. it gets bundled exactly as is into the app.

2

configure your app

set the app name, package identifier, and version. these get written into the android manifest and signing config.

3

build and download

hit the build button. your html gets wrapped in a webview, compiled into a debug apk, signed with a keystore, and served for download.
